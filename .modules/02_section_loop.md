Change Log entry - Experimental branch #1: Added and adjusted for short and detailed summary modes
### **Module 2 — Section Summarizer**

Summary Modes:
- **summary_level = "short"**
- **summary_level = "detailed"**

Loop through each section:

- **Extract text**  

- **if summary_level = "detailed" create a short paragraph **plus** a bullet list of 3–5 key points for each section**

- **if summary_level = "short" create a 1–2 sentence summary per section**

- **Adjust language to match user profile** (simplified for beginners, technical for experts)  

- **Mark if section is missing or empty**
